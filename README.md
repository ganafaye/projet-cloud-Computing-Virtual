# Projet Cloud Computing - Architecture 3-Tiers

## Description
Déploiement automatisé d'une infrastructure 3-tiers sur OpenShift Virtualization.

## Architecture
- **VM-GW** : Passerelle/Firewall (iptables)
- **VM-WEB** : Serveur Web (Nginx + Node.js)
- **MySQL** : Base de données (Pod)
